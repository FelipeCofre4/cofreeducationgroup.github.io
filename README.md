# COFRÉ EDUCATION GROUP - Página Web

## 📱 Descripción

Página web profesional, moderna y responsiva para COFRÉ Education Group SpA. Un sitio completo que integra la identidad de marca, servicios y opciones de contacto.

## ✨ Características

### Diseño
- ✅ **Responsivo** - Funciona perfectamente en móvil, tablet y desktop
- ✅ **Moderno y Profesional** - Diseño corporativo acorde a tu marca
- ✅ **Colores Personalizados** - Gradiente verde-azul del logo
- ✅ **Animaciones Suaves** - Transiciones fluidas y fade-in effects

### Secciones
1. **Header Sticky** - Navegación que se queda visible al desplazarse
2. **Hero Section** - Impacto visual con propuesta de valor
3. **Sobre Nosotros** - Tu perfil como fundador y valores de la empresa
4. **Servicios** - Tarjetas interactivas de los 3 servicios principales
5. **Casos de Éxito** - Portfolio de proyectos realizados
6. **Contacto** - Formulario integrado + información de contacto
7. **Footer** - Links e información corporativa

### Funcionalidades
- 📱 **Menú Móvil** - Menú hamburguesa para dispositivos pequeños
- 🎯 **Scroll Suave** - Navegación fluida entre secciones
- 📊 **Animaciones en Scroll** - Los elementos se animan al entrar en vista
- 📝 **Formulario de Contacto** - Integrado con WhatsApp y correo
- 🔗 **Enlaces Optimizados** - Teléfono y correo clicables

### Modificar Servicios
Dentro de la sección `<!-- SERVICIOS -->`, puedes:
- Cambiar títulos y descripciones
- Agregar/quitar características
- Modificar emojis de los iconos

### Actualizar Casos de Éxito
En la sección `<!-- PORTAFOLIO -->` puedes:
- Cambiar títulos de proyectos
- Actualizar descripciones
- Agregar nuevos casos

### Cambiar Colores
Busca `:root` al inicio del CSS y modifica:
```css
--primary-dark: #1e3a5f;      /* Azul oscuro */
--primary-blue: #2b7fbf;      /* Azul principal */
--accent-green: #4db8a8;      /* Verde/turquesa */
```
